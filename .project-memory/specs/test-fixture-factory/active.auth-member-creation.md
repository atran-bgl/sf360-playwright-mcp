# Spec: Member Creation (3-Step Process)

**Feature:** Create member via SF360 API using 3-step process
**Priority:** High
**Estimated Complexity:** High

---

## Overview

Member creation in SF360 requires a **3-step process**:
1. **Create Contact** (person entity) → Returns `peopleId`
2. **Retrieve Member Data** by peopleId → Returns `memberCode`
3. **Create Accumulation Account** using memberCode → Returns `memberId`

**Source:** `noncompliance20260116/tests/lib/member-util.js` and `contact-util.js`

---

### Options Parameter

```typescript
interface CreateMemberOptions {

```

### Return Value

```typescript
interface MemberResult {
  memberId: string;        // Member account ID (from step 3)
  memberCode: string;      // Member code (from step 2)
  memberName: string;      // Full name (firstName + lastName)
  peopleId: string;        // Contact/person ID (from step 1)
}
```

## Step 1: Create Contact (Person)

### Endpoint

```
POST ${baseUrl}/entity/mvc/base/addPeople?firm=${firm}&uid=${uid}
Content-Type: application/x-www-form-urlencoded
```

### Payload

```javascript
const contactPayload = {
  allocationId: null,
  peopleType: 'person',
  entitySubtype: null,
  firm: firm,
  title: options.sex === 'Male' ? 'Mr' : 'Ms',
  entityType: 'person',
  entitySubType: 'person',
  firstName: options.firstName || 'Test',
  middleName: '',
  surname: options.lastName || `Member${Date.now()}`,
  dob: options.dateOfBirth || '1980-01-01',
  sex: options.sex || 'Male',
  countryOfBirth: '',
  tfn: options.tfn || '',
  email: options.email || '',
  mobile: options.mobile || '',
  homePhone: '',
  workPhone: '',
  addressLine1: '',
  addressLine2: '',
  suburb: '',
  state: '',
  postcode: '',
  country: 'Australia',
  isCustom: false
};

// Convert to URL-encoded format
const formData = new URLSearchParams(contactPayload).toString();

const response = await axios.post(
  `${baseUrl}/entity/mvc/base/addPeople?firm=${firm}&uid=${uid}`,
  formData,
  {
    headers: {
      'Content-Type': 'application/x-www-form-urlencoded'
    }
  }
);

const peopleId = response.data.id;
```

## Step 2: Retrieve Member Data

### Endpoint

```
GET ${baseUrl}/entity/mvc/base/getContacts?firm=${firm}&uid=${uid}&fundId=${fundId}
```

### Process

```javascript
// Fetch all contacts for the fund
const contactsResponse = await axios.get(
  `${baseUrl}/entity/mvc/base/getContacts?firm=${firm}&uid=${uid}&fundId=${fundId}`
);

const contacts = contactsResponse.data;

// Find the contact we just created by peopleId
const contact = contacts.find(c => c.id === peopleId || c.peopleId === peopleId);

if (!contact) {
  throw new Error(`Contact not found: peopleId=${peopleId}`);
}

// Get detailed person info
const personResponse = await axios.get(
  `${baseUrl}/entity/mvc/base/getPersonDetails?firm=${firm}&uid=${uid}&fundId=${fundId}&peopleId=${peopleId}`
);

const personDetails = personResponse.data;

// Extract member code (generated by SF360)
const memberCode = personDetails.memberCode || personDetails.code;

if (!memberCode) {
  throw new Error('Member code not found in person details');
}
```

## Complete Implementation

```javascript
const axios = require('axios');

/**
 * Create member via 3-step SF360 API process
 * @param {Object} options - Member creation options
 * @returns {Promise<Object>} { memberId, memberCode, memberName, peopleId }
 */
async function createMember(options) {
  const {
    firm,
    uid,
    fundId,
    baseUrl,
    firstName = 'Test',
    lastName,
    dateOfBirth = '1980-01-01',
    sex = 'Male',
    email = '',
    mobile = '',
    tfn = ''
  } = options;

  if (!fundId) {
    throw new Error('fundId is required to create member');
  }

  // Generate unique last name if not provided
  const memberLastName = lastName || `Member${Date.now()}`;
  const fullName = `${firstName} ${memberLastName}`;

  try {
    console.log(`Creating member: ${fullName}...`);

    // STEP 1: Create Contact (Person)
    const contactPayload = {
      allocationId: null,
      peopleType: 'person',
      entitySubtype: null,
      firm: firm,
      title: sex === 'Male' ? 'Mr' : 'Ms',
      entityType: 'person',
      entitySubType: 'person',
      firstName: firstName,
      middleName: '',
      surname: memberLastName,
      dob: dateOfBirth,
      sex: sex,
      countryOfBirth: '',
      tfn: tfn,
      email: email,
      mobile: mobile,
      homePhone: '',
      workPhone: '',
      addressLine1: '',
      addressLine2: '',
      suburb: '',
      state: '',
      postcode: '',
      country: 'Australia',
      isCustom: false
    };

    const formData = new URLSearchParams(contactPayload).toString();

    const contactResponse = await axios.post(
      `${baseUrl}/entity/mvc/base/addPeople?firm=${firm}&uid=${uid}`,
      formData,
      {
        headers: {
          'Content-Type': 'application/x-www-form-urlencoded'
        }
      }
    );

    const peopleId = contactResponse.data.id;

    if (!peopleId) {
      throw new Error('Failed to create contact: No peopleId returned');
    }

    console.log(`  ✓ Contact created (peopleId: ${peopleId})`);

    // STEP 2: Retrieve Member Data
    // Wait a moment for SF360 to process the contact
    await new Promise(resolve => setTimeout(resolve, 500));

    const personResponse = await axios.get(
      `${baseUrl}/entity/mvc/base/getPersonDetails?firm=${firm}&uid=${uid}&fundId=${fundId}&peopleId=${peopleId}`
    );

    const personDetails = personResponse.data;
    const memberCode = personDetails.memberCode || personDetails.code;

    if (!memberCode) {
      throw new Error('Failed to retrieve member code from person details');
    }

    console.log(`  ✓ Member data retrieved (memberCode: ${memberCode})`);

    // STEP 3: Create Accumulation Account
    const accountPayload = {
      id: null,
      memberCode: memberCode,
      peopleId: peopleId,
      accountName: `${firstName} ${memberLastName} - Accumulation`,
      accountType: 'Accumulation',
      status: 'Active',
      openingDate: new Date().toISOString().split('T')[0],
      closingDate: null,
      balance: 0,
      taxComponents: {
        taxable: 0,
        taxFree: 0
      }
    };

    const accountResponse = await axios.post(
      `${baseUrl}/chart/chartmvc/MemberController/save?firm=${firm}&uid=${uid}&fundId=${fundId}`,
      accountPayload,
      {
        headers: {
          'Content-Type': 'application/json'
        }
      }
    );

    const memberId = accountResponse.data.id || accountResponse.data.accountId;

    if (!memberId) {
      throw new Error('Failed to create accumulation account: No memberId returned');
    }

    console.log(`✓ Member created: ${fullName} (memberId: ${memberId})`);

    return {
      memberId,
      memberCode,
      memberName: fullName,
      peopleId
    };

  } catch (error) {
    if (error.response) {
      throw new Error(
        `Failed to create member: ${error.response.status} - ${JSON.stringify(error.response.data)}`
      );
    } else if (error.request) {
      throw new Error('Failed to create member: No response from server');
    } else {
      throw new Error(`Failed to create member: ${error.message}`);
    }
  }
}

module.exports = { createMember };
```

## Integration with setupTest()

```javascript
const { createFund } = require('./fund-api');
const { createMember } = require('./member-api');

async function setupTest(page, options) {
  const { firm, pageKey } = options;
  const config = loadConfig();

  // ... authentication ...

  // Create fund if needed
  if (requiresFund) {
    const fundResult = await createFund({
      firm,
      uid: config.uid,
      baseUrl: config.baseUrl,
      name: options.fundName
    });

    fundId = fundResult.fundId;
    fundName = fundResult.fundName;
  }

  // Create member if needed (3-step process)
  if (requiresMember) {

      throw new Error('Cannot create member without fund');

      firm,
      uid: config.uid,
      fundId,
      baseUrl: config.baseUrl,

    memberId = memberResult.memberId;
    memberCode = memberResult.memberCode;
    memberName = memberResult.memberName;
    peopleId = memberResult.peopleId;
  }

  return {
    baseUrl: config.baseUrl,
    firm,
    uid: config.uid,
    fundId,
    fundName,
    memberId,
    memberCode,
    memberName,
    peopleId
  };
}
```

## Error Handling

### Step 1 Errors (Contact Creation)

```javascript
// Error: Duplicate contact
{
  "error": "Contact already exists with this name",
  "code": "DUPLICATE_CONTACT"
}

Fix: Use unique names (timestamp-based)
```

### Step 2 Errors (Member Data Retrieval)

```javascript
// Error: Contact not found
Error: Contact not found: peopleId=8a8bc49f...

Fix:
1. Wait 500ms after contact creation
2. Verify peopleId is correct
3. Check fund has access to contact
```

### Step 3 Errors (Accumulation Account)

```javascript
// Error: Invalid member code
{
  "error": "Member code not found",
  "code": "INVALID_MEMBER_CODE"
}

Fix:
1. Verify memberCode from step 2
2. Ensure step 2 completed successfully
3. Check fundId is correct
```

## Timing Considerations

### Delays Between Steps

SF360 may need processing time between steps:

```javascript
// After contact creation
await new Promise(resolve => setTimeout(resolve, 500));

// After member data retrieval (optional)
await new Promise(resolve => setTimeout(resolve, 200));
```

**Recommended:**
- After Step 1: 500ms delay
- After Step 2: No delay needed (synchronous)

- Each member can have multiple accounts
